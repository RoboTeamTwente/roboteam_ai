bool rtt::ai::stp::Play::shouldEndPlay ( )
protectedvirtualnoexcept

This function is used to determine if – when a role is in an endTactic – the endTactic should be skipped. An example could be BlockRobot and Intercept. You block a robot (endTactic) until a ball is shot and then the robot closest to the ball should try to intercept (skip the BlockRobot tactic to execute Intercept) Optional function to force end plays

Returns
True if play should end this tick
