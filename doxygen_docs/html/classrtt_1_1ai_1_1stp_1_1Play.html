Detailed Description

Play class that's used in the STP model on update traverses every Role, and updates it.

This function is used to determine if – when a role is in an endTactic – the endTactic should be skipped. An example could be BlockRobot and Intercept. You block a robot (endTactic) until a ball is shot and then the robot closest to the ball should try to intercept (skip the BlockRobot tactic to execute Intercept) Map that holds info from the previous play
