std::optional<StpInfo> MockTactic::calculateInfoForSkill ( const StpInfo &  info)
inlineoverridevirtualnoexcept

This function should calculate any extra information that the skills might need to be executed. Things that should be calculated are for example how hard the kicker should shoot to get to the desired position or how fast the dribbler should be spinning.

Though this method is responsible for ensuring everything is calculated, it helps to use helpers so this function doesn't become a massive hack
